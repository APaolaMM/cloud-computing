Transcripción Computación en la Nube

¡Saludos!

Les doy la bienvenida al tema “Cloud Computing”, que consiste en un modelo de computación que permite al proveedor tecnológico ofrecer servicios informáticos a través de internet. 

Algunas de sus características son: 

  • Pago por uso.

  • Acceso desde la red.

  • Recursos compartidos.

  • Recursos a la carta o escalabilidad.

  • Servicio supervisado.

Cuenta con bastantes ventajas, entre ellas:

  • Reducción de costes.

  • Optimización de recursos.

  • Recuperación ante desastres.

  • Tecnología actualizada y segura.

  • Dedicación al negocio.

Por otra parte, implica ciertos inconvenientes, citados a continuación:

  • Pérdida de control, ya que no se tiene acceso a las instalaciones.

  • Confidencialidad y seguridad en los datos, ya que se almacena en los servidores del proveedor.

  • Riesgo de que se caiga.

  • No se puede acceder sin acceso a internet.

Algunas opciones de contratación son:

  • SAAS (Software As A Service) software como servicio: Se entrega el software dentro de sus instalaciones para el uso a través de internet siempre que lo demande el usuario.

  • PAAS (Platform As A Service) plataforma como servicio: Se entrega una plataforma con el hardware, sistema operativo y middleware o las API para que el cliente desarrolle un servicio o aplicación.

  • IAAS (Infraestructure As A service) infraestructura como servicio: Se entrega el acceso a la infraestructura de computación bajo demanda.

Por último, se muestran algunos riesgos de este modelo:

  • Acceso de usuarios con privilegios

  • Incumplimiento normativo

  • Desconocimiento de la localización de los datos

  • Falta de aislamiento de los datos

  • Indisponibilidad del servicio en caso de un desastre o incidente

  • Carencia de soporte investigativo

  • Viabilidad a largo plazo
